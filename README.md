# INANI — App Demo

**Put a number on it.** Sports intelligence for South Africa.

Static demo build for partner and investor previews. No backend, no build step.

## Host on GitHub Pages
1. Create a repo and upload **both** `index.html` and the entire **`assets/`** folder (keep the folder name — the photography is referenced as `assets/*.jpg`).
2. Settings → Pages → Deploy from branch → root.
3. Your demo is live at `https://<user>.github.io/<repo>/`.

## What this demo shows
- Full-bleed match-day photography behind every screen, rotating hero on the front door
- Registration → package selection → **sandbox checkout** (clearly labelled; no real payment)
- Entitlement-gated views: Free / Athlete Pro R99 / Athlete Elite R299 / Team Pro / Team Elite / Brand Premium R4,999
- Rate cards stay private at every tier except your own; Brand Premium sees cohort value corridors only
- Simulated read-only OAuth (POPIA-framed), rand valuation engine anchored to a 2022 Springbok PR rate card, Inani Insights newsroom, deals marketplace
- **Demo controls** on the Account screen switch packages instantly for walkthroughs
